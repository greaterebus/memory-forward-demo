# Memory Forward — Internal Knowledge Base

Welcome to Memory Forward's internal repository. This is the single source of truth for our
operational processes, automation scripts, and documentation standards.

## What We Do

Memory Forward digitizes, restores, and organizes physical media — printed photos, film negatives,
35mm slides, VHS/Beta tapes, 8mm film, and more — returning customers' memories as organized
digital archives they can access forever.
